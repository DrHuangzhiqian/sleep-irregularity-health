# sleep-irregularity-health
This repository provides the analytical code and reproducible scripts underlying a comprehensive investigation of accelerometer-derived sleep irregularity and its relevance to human health and disease. Using large-scale data from the UK Biobank, the study systematically characterizes sleep irregularity in relation to multi-system clinical traits, incident disease outcomes, multi-omic signatures, genetic architecture, and potential causal pathways. The analytical framework integrates wearable-derived sleep phenotyping, prospective epidemiological modelling, external replication, metabolomic and proteomic profiling, mediation analyses, genome-wide association analyses, Mendelian randomization, and longitudinal trajectory-based quasi-interventional analyses. Collectively, these scripts support a multidimensional assessment of sleep irregularity as a biologically embedded and clinically meaningful dimension of sleep health beyond sleep duration.
